## How to use this file

Add one line per issue found during proofing. Format:

```
p. N — "Poem Title": description of issue
```

For layout issues (widows, orphans, bad page breaks):
```
p. N — layout: last line of "Poem Title" stanza stranded at page top
```

For footnote issues:
```
p. N — "Poem Title": footnote mark N orphaned — text not on same page
```

For content issues:
```
p. N — "Poem Title" l. 5: word broken incorrectly / line missing / wrong text
```
