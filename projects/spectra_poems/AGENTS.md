# spectra_poems — Project AGENTS

The tracked **example** project (*Iberian Dreams*, a few poems) — the reference
others can be modeled on, and the one project whose source is committed as a
buildable demonstration. Not part of the Fletcher series.

This file closes the loop for routine work; for framework/module reference see the
repo `README.md` § Agent Framework.

## Commands

```
texgraph proof-build      --project spectra_poems
texgraph proof-preview    --project spectra_poems
```

## Notes

Small, illustrative content under `manuscript/reading/` and a single
`interior/collection.yaml`. Same pipeline and conventions as the Fletcher volumes
(see `projects/fletcher-early-works/AGENTS.md`); keep it minimal — it exists to
demonstrate a working project end to end. Never hand-edit `interior/output/`.
