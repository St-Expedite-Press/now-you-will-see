## Method

- Render pages with `.\.venv\Scripts\texgraph.exe pdf render`.
- Use OCR for navigation only; the source PDF has poor OCR quality.
- Irradiations have no individual titles — use `title: "Irradiation I"` etc.
- The roman numeral heading on the scan page is the title. If a scan page
  has two roman-numeral headings (short poems sharing a page), record both
  as separate files with the same `source_pages_scan`.
- Epilogue has the title "Epilogue" and belongs in `book_part: "Irradiations"`.
- For Sand and Spray movements, use `book_part: "Sand and Spray"` and
  `series: "Sand and Spray"` with `series_part` set to the movement name.
- Set `status: "transcribed"` only after visual image checking.
