# Raw Source Files

Raw source PDFs are preserved here for source control. Do not edit, optimize, or
overwrite these PDFs. If a replacement source is needed, add it under a new,
stable filename and update `metadata/source_manifest.md`.

## Layout

- `early_1913/`: the five 1913 books for _The Early Works of John Gould
  Fletcher_.
- `middle_1915_1928/`: acquired public-domain middle-period Fletcher books.
- `late_rights_pending/`: late books with cleared permission; source PDFs still
  to be acquired. Do not store restricted lending files here.
